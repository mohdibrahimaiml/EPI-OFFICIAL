Verification

Five independent checks.
Zero blind trust.

No server. No upload. No installation. Just a browser and a .epi file.

01 Structure Validation

Valid envelope, extractable ZIP, all required files present. Every byte in its designated place.

✓ STRUCTURE_VALID

02 Hash Integrity

Every file hash in file_manifest matched against computed SHA-256. One mismatch triggers failure.

✓ HASHES_MATCH

03 Signature Verification

Ed25519 signature validated against canonical manifest hash. Verifiable with only the public key.

✓ SIGNATURE_VALID

04 Chain Continuity

prev_hash links intact. Monotonic indices preserved. Timestamps never decrease. One timeline.

✓ CHAIN_INTACT

05 SCITT Anchoring

COSE_Sign1 statement and transparency receipt verified against the ledger. Publicly auditable, independently verifiable.

Integration

Three lines of Python.
Signed. Sealed. Shipped.

No monkey patching. Works with streaming and non-streaming. Compatible with any LLM SDK.

agent.py
from openai import OpenAI from epi_recorder import record, wrap_openai client = wrap_openai(OpenAI()) with record("agent.epi", goal="Assess loan application"): response = client.chat.completions.create( model="gpt-4", messages=[{"role":"user", "content":"Assess this"}] ) # Every call → signed .epi evidence # $ epi verify agent.epi → VERIFIED ✓
