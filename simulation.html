EPI Workflow Simulator

Experience the complete evidence lifecycle: Install, Record, Verify, View.

1. Install
Installs the epi-recorder package from PyPI. Zero system dependencies required.
2. Record
Explicitly captures inputs, API calls (OpenAI), and artifacts using log_llm_call() or wrappers.
3. Verify
Checks cryptographic signatures (Ed25519) and file integrity hashes.
4. View
Opens the self-contained evidence package in the browser viewer.
